Описание проекта
Этот проект представляет собой учебный фан-сайт, посвященный легендарной американской рок-группе Linkin Park. Сайт создан для отработки навыков HTML-верстки и содержит полную информацию о группе, включая историю, дискографию, фильмографию и возможность "заказа" товаров.

Структура проекта
HTML-страницы
Index.html - Главная страница с введением

Info.html - Общая информация о группе, составе и достижениях

History.html - Подробная история группы с хронологией событий

Disc.html - Полная дискография с альбомами и треками

Film.html - Фильмография и саундтреки

Links.html - Источники информации и библиография

Shop.html - Страница "заказа" товаров

Contact.html - Контакты и форма обратной связи

film1.jpg - Постер Collision Course

film2.jpg - Постер Road to Revolution

film4.jpg - Постер The Hunting Party Tour

film5.jpg - Постер One More Light Live

poster_set.jpg - Набор постеров альбомов

Особенности сайта
Дизайн и стиль
Единая цветовая схема: темный фон (#0a0a0a, #16213e, #1a1a2e) с акцентным цветом #1DB954

Единая навигационная панель на всех страницах

Адаптивные таблицы и формы

Стилизованные разделители и элементы

Функциональность
Полная навигация между всеми страницами

Формы обратной связи и заказа (учебные, не функциональные)

Внешние ссылки на официальные ресурсы

Внутренние якорные ссылки для навигации по длинным страницам

Технические детали
Используемые технологии
Чистый HTML (без CSS файлов, стили inline)

Базовые HTML-элементы: таблицы, формы, списки, изображения

Ссылки, якоря, мета-теги

Особенности верстки
Использование устаревших атрибутов (align, bgcolor, border)

Inline-стилизация

Семантическая разметка с использованием заголовков h1-h3

Табличная верстка для некоторых разделов
